响应组件内用户的文本选择

[⬆ Back to Top](#table-of-contents)

## Features

1. 一个提供监听组件内用户文本选择功能的组件，通过 slot 与其他组件组合
2. 提供两个定位于文本选择位置 slot，方便在出现文本选择时，在相应位置作出 UI 上的响应
